<h1>Bible Reading Website</h1>
<p>A simple, responsive website for daily Bible reading plans, built with plain HTML, CSS, and JavaScript.</p>

<h2>Pages</h2>
<p><strong>M'Cheyne Reading Plan</strong> (<code>index.html</code>) — Daily readings from Robert Murray M'Cheyne's plan, split into Family and Secret categories.</p>
<p><strong>LFBC 2026 Chronological Plan</strong> (<code>lfbc.html</code>) — Daily readings from Living Faith Baptist Church's 2026 chronological Bible reading plan.</p>
<p><strong>Full Calendars</strong> (<code>calendar.html</code>, <code>lfbc-calendar.html</code>) — Complete year-at-a-glance tables for each plan with today's row highlighted.</p>

<h2>Features</h2>
<ul>
  <li>Light and dark mode toggle (persisted via localStorage)</li>
  <li>Optional woodcut background image toggle</li>
  <li>Decorative ornament displayed when background is off</li>
  <li>Navigation between days with a "Return to today" button</li>
  <li>Mobile-friendly with tightened spacing on smaller screens</li>
  <li>No frameworks or build tools — just static HTML files</li>
</ul>
